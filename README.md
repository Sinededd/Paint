# Графический редактор геометрических фигур
## Description
Графический редактор геометрических фигур написанный на Qt Widgets.
В приложении можно создавать примитимные геометрические фигуры: прямоугольник, эллипс, треугольник, многоугольник, и звезду.
В фигурах можно изменить размер, поворот относительно цетра, и некоторые другие параметры.
Также можно выделять и перемещать фигуры, делать множественное выделение, удалять по клавише Del.
